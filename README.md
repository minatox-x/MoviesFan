## What is Smarter?

Smarter is a native Android streaming companion that lets you search any movie or TV show, browse rich metadata, and watch directly inside the app вҖ” no browser, no redirects, no nonsense. It wraps a smart WebView player that blocks ads at the navigation layer while letting the actual video through cleanly.

## How the Player Works

The player is intentionally minimal:

```kotlin
private fun loadPlayerFrame(embedUrl: String) {
    playerWebView.loadUrl(embedUrl)
}
```

That's it. No wrapper HTML, no fake base URLs, no iframe injection. The embed URL loads directly. Ad blocking happens entirely inside `SmartWebViewClient.shouldOverrideUrlLoading`:

```
Request comes in
    в”Ӯ
    в”ңв”Җ Blocked scheme? (intent://, market://, tel://)  вҶ’  DROP
    в”ңв”Җ Known ad host? (doubleclick, play.google.com)   вҶ’  DROP
    в”ңв”Җ Known player host?                              вҶ’  ALLOW вң“
    в”ңв”Җ Originated from a player host? (Referer check)  вҶ’  ALLOW вң“
    в””в”Җ Everything else                                 вҶ’  DROP
```

Player redirect chains (vidsrc вҶ’ cloudnestra вҶ’ rabbitstream вҶ’ CDN) all flow through because each hop is either a known player domain or carries a player referer.
